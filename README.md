# Tempalert-iot
Real-time temperature monitoring and alert system for small animals

## Components
- Temperature sensor (MLX90614-GY906-Module)
- Arduino-compatible microcontroller (ESP32WROOM-32CP)
- OLED display
- Transistor (LM7805T)
- Capacitor (470uf)


## Files
- `schematic.pdf` — circuit diagram
- `pcb_layout.pdf` — PCB layout
- `gerber.zip` — fabrication files (if applicable)
- `images/` — photos of the fabricated PCB
